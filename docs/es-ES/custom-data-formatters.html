Formato de datos

ANTES DE COMENZAR

  1. Para utilizar un formato personalizado, lo primero que deberá hacer es crear una carpeta YourDatabase.4dbase/Resources/Mobile/formatters.

  2. Luego suelte la carpeta del formato en ella.
